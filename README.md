# View in browser plus

An improved version of sublime view in browser that allows for multiple preview entries

## Installation

1. Copy the whole package folder to your packages folder (Preferences>Browse Packages)
2. Move the settings file into `Packages/User`

In the package folder, the files Main.sublime-menu and ViewInBrowserPlus.py are required. The others can be removed.

Edit your preferences either through `Preferences > Package Settings > View in Browser Plus` or just in the settings file and add preview entry mappings.

## Key Bindings

Add the following to your key bindings

```
{ "keys": ["f12"], "command": "view_in_browser_plus"}
```
